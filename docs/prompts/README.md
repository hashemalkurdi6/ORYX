# Prompts

Saved Claude Code prompts that produced good results, kept here so we don't have to re-derive them.

## Conventions

Each prompt file has a header with:

- **What it does** — the outcome to expect.
- **When to use** — the trigger / situation it fits.
- **Gotchas** — anything that broke the first time, edits we made, files it should NOT touch.

The actual prompt body goes below the header, ready to copy-paste into Claude Code.

## Categories

- `master-9-week-plan.md` — the launch execution plan.
- `audit-prompt.md` — the auditor invocation we use across the 5 audit agents.
- `pre-testflight-checklist.md` — last-mile checks before pushing a TestFlight build.

Add new prompts as their own files. If a category grows past ~5 prompts, split it into a subfolder.
